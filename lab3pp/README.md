# Лабораторная работа №3  
## Параллельное программирование (MPI)

---

### Цель работы  
Исследовать зависимость времени выполнения программы умножения матриц от размера матриц и количества процессов.

---

### Задание  
Модифицировать программу умножения матриц с использованием технологии MPI. Провести эксперименты при различных размерах матриц и количестве процессов.

---

### Описание программы  
Программа выполняет умножение квадратных матриц. Матрицы генерируются автоматически в коде.  
Для распараллеливания используется технология MPI:  
- данные распределяются между процессами с помощью `MPI_Scatter`  
- результаты собираются через `MPI_Gather`  
- используется широковещательная передача `MPI_Bcast`  

Количество процессов задаётся при запуске программы.

### Анализ результатов  

С увеличением размера матрицы время выполнения программы увеличивается.  
При использовании нескольких процессов наблюдается ускорение вычислений.  
Однако при большом количестве процессов прирост производительности снижается из-за накладных расходов на обмен данными между процессами.  

---

### Вывод  

В ходе лабораторной работы была реализована параллельная программа умножения матриц с использованием технологии MPI.  

Проведённые эксперименты показали, что время выполнения зависит от размера матриц и количества процессов. Использование параллельных вычислений позволяет ускорить выполнение программы, особенно при больших размерах данных.
