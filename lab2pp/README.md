Лабораторная работа №2  
Параллельное программирование (OpenMP)

Цель работы  
Исследовать зависимость времени выполнения программы умножения матриц от размера матриц и количества потоков.

Задание  
Модифицировать программу умножения матриц с использованием OpenMP. Провести эксперименты при различных размерах матриц и количестве потоков.

Описание программы  
Программа выполняет умножение квадратных матриц. Матрицы генерируются автоматически в коде. Для распараллеливания используется директива OpenMP parallel for. Количество потоков задаётся пользователем.

Анализ результатов:  
С увеличением размера матрицы время выполнения программы увеличивается. При использовании нескольких потоков наблюдается ускорение вычислений. При малых размерах матриц ускорение незначительное из-за накладных расходов на создание потоков. При увеличении размера задачи эффективность параллелизации возрастает.

Вывод:  
В ходе лабораторной работы была реализована параллельная программа умножения матриц с использованием OpenMP. Проведённые эксперименты показали, что время выполнения зависит от размера матриц и количества потоков. Использование параллельных вычислений позволяет ускорить выполнение программы при увеличении объёма данных.
